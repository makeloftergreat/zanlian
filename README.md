# ♥ 赞链 ZanLian

> LOFTER 点赞互助平台 — 先付后得，公平排队，双账号隔离保护隐私

**当前版本：v0.4.0**

---

## 简介

赞链是一个基于 LOFTER 平台的点赞互助工具。用户通过"马甲号"为他人点赞，自己的"创作号"排队等待回赞。马甲号和创作号完全隔离，保护你的创作号隐私。

## 在线体验

👉 [https://zanlian.vercel.app](https://zanlian.vercel.app)

## 核心机制

| 机制 | 说明 |
|------|------|
| **先付后得** | 你必须先给他人点赞，自己的创作号才能进入排队队列 |
| **先来后到** | 排队按加入顺序，位置透明可查，排在前面先获得回赞 |
| **溯源日志** | 每次点赞都有记录，格式为"马甲A -> 马乙"，只显示马甲号 |
| **双账号隔离** | 马甲号用于点赞（可见），创作号用于收赞（仅自己可见） |
| **每日上限** | 每天最多点赞 20 次，防止滥用，0 点自动重置 |
| **点赞验证** | 系统自动抓取文章页点赞列表，验证你是否真的点了赞 |

## 使用流程

1. **注册** — 填写马甲号、创作号、邮箱和密码
2. **领任务** — 系统自动分配一个待点赞的文章
3. **去点赞** — 用马甲号打开文章，点红心
4. **确认** — 回到赞链点击"确认完成"，系统自动验证点赞
5. **排队** — 验证通过后，你的创作号自动排入队列
6. **收赞** — 轮到你时，他人会给你点赞

## 技术栈

- **前端**：Vue 3 (CDN) + 原生 CSS 深色主题
- **后端**：Supabase (Auth + REST API + PostgreSQL)
- **部署**：Vercel（自动部署，0 费用）

## 安全保障

- **每日上限**：每天最多点赞 20 次，防止滥用
- **密码安全**：密码由 Supabase Auth 加密存储，赞链无法看到明文密码
- **隐私保护**：创作号信息仅自己可见，溯源日志中不出现任何创作号
